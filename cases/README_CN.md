# 基准案例

基准案例基于B size的网格模型。

## 稳态案例

在运行瞬态案例前，推荐运行稳态案例`steady`和`TFM-steady`。这两个案例基于SIMPLE计算了稳态的空气流场，并能够映射至瞬态案例的初始条件，加速瞬态案例的收敛。其中：

- `steady`是面向`STM-1`/`STM-2`/`STM-3`/`MM`的稳态案例
- `TFM-steady`是面向`TFM`的稳态案例


## 瞬态案例

瞬态案例的名称和论文中的五个方法名称一致，分别是：

- `STM-1`：基于标量输运模型和经验跃移层
- `STM-2`： 基于标量输运模型和湍流耗散源项
- `STM-3`：基于标量输运模型和冠层源项
- `MM`：基于混合物模型
- `TFM`：基于双流体模型


## 运行案例

进入案例目录，例如`STM-1`：

```bash
cd STM-1
```

运行串行案例（推荐）：

```bash
./Allrun
```

运行并行案例：

```bash
./Allrun-parallel
```
